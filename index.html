noSQL (not Structured Query Language)

not RDBMS ?

  • Uses JavaScript (provides drivers for other languages)
  • Dynamic schema
  • no relation (we can imitate it)
  • horizontal and vertical scalability
  • works fast even with big data
  • mongoDB - 2009
